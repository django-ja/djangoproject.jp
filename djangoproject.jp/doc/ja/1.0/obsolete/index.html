撤廃された機能のドキュメント¶

revision-up-to:8961 (1.0)

このセクションのカバーしている機能は、現在の Django ではすでに撤廃されたか、 別の機能にとって代わられています。ここに残しているのは、古いバージョンの Django を使っているユーザや、撤廃された API を使っているユーザのためです。 新しくコードを書くときには、このセクションにある API を使わないでください。
