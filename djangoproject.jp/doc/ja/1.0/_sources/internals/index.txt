Django の内部
================

:revision-up-to: 8961 (1.0)

ここにあるのは、 Django 自体をハックする人達のためのドキュメントです。
Django の改善を手伝ったり、 Django が舞台裏でやっていることを学んだりしたけ
れば、ここにあるドキュメントを読みましょう。

.. warning::

    Django ドキュメントの他のセクションでは、書かれている内容に一定の保証が
    あります。すなわち、ある API がひとたび公式のドキュメントとして公開され
    たら、その API は「安定」であるとみなされ、十分妥当な理由がない限り変更
    されません。しかし、ここで扱っている API は「内部向け」であり、必要であ
    れば変更する余地を残しています。
