Django 1.6 alpha 1 リリースのお知らせ

  • 4 years, 2 months ago
  • Hiroki Kiyohara
    • 0 comments

以下の文章は本家ブログ記事の Django 1.6 alpha 1 released を訳したものです。

Django 1.6 リリースプロセスの一環として、本日(5月26日に) Django 1.6 alpha 1 がリリースされました。 これはプレビュー/テスト用パッケージで、 Django 1.6 で新しくなる点を試せます。 他の alpha, beta パッケージと同じように、この Django 1.6 alpha 1 もプロダクションに使うものでは ありません。 Django 1.6 の新しい良い点を試してみたい、 来る Django 1.6 リリース (8 月予定) の前にバグフィックスを助けたいというのであれば、 Django 1.6 のコピーを手に入れて走らせてみて ください。

Django 1.6 alpha パッケージはこの ダウンロードページ で入手できます。 その際には リリースノート を読んでおくのが良いでしょう。 セキュリティ面を考慮して、Django 1.6 alpha パッケージの MD5 と SHA1 チェックサム が 用意されています。

このスケジュールが意味するのは、この夏に Django 1.6 がリリースされると、リリースされてからまだ2年も経っていない Django 1.4 のサポートが終わるということです(リリースポリシーをよく読んでいる人は気づいてるかもしれません)。 まだ詳細は決まっていませんが、 Django 1.4 は広く使われているのでサポートは 多少延長します。詳細の決定に協力してくださるなら、議論は django-developers メーリングリストで行われていますので参加してください。
